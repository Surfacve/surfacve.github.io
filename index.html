2025-04-06 Project

Building this site: a pleasantly simple start

Set up GitHub Pages today. The whole thing took about 20 minutes — create a repo, enable Pages in Settings, drop in an index.html. That's it.

The thing I keep coming back to: resist the urge to over-engineer. Plain HTML is fast, readable, and needs no build step. Future me will thank present me for this.

"The best tool for a job is often the one that gets out of your way."
